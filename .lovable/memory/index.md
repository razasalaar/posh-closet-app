# Project Memory

## Core
Pakistani fashion eCommerce "LUXE". White luxury theme, gold accent, Playfair Display headings, Inter body.
Zustand for cart/wishlist state. Prices in PKR (Rs. format).
Lovable Cloud backend (Supabase). Auth with email/password auto-confirm.
Admin role-based access via user_roles table + has_role() security definer function.
